INTERPRETATION (appended by i8_daily_risk.py author)
================================================================================
THESIS CONFIRMED, AND THE UNDERSTATEMENT IS LARGE.

Monthly NAV marks (I2) understated the TRUE intra-month max-drawdown in every
regime, catastrophically so in the crash:

  regime               monthly_DD   TRUE daily_DD   understatement   trough (daily)
  crash_2020             -2.6%        -20.56%         7.97x  (-18.0pp)  2020-03-23
  bear_2022              -6.6%        -10.31%         1.56x  ( -3.7pp)  2022-03-07
  recovery_2023_2024     -0.8%         -4.03%         4.80x  ( -3.2pp)  2023-10-27

* MONTHLY-MARK REPRODUCTION IS EXACT: this driver's monthly_max_dd_pct_reproduced
  (-6.60 / -2.58 / -0.84) and wheel_return_pct (4.49 / 9.0 / 40.2) reproduce I2's
  published figures, so the daily series shares I2's exact position book.

* THE SMOKING GUN IS 2020. I2's monthly mark caught the book on 2020-04-01, AFTER
  the V-recovery had begun, so it logged only -2.6%. The DAILY path shows the book
  fell -20.56% peak(2020-02-24)->trough(2020-03-23) -- the exact COVID bottom -- then
  recovered by the next snapshot. The monthly mark hid ~18pp of realized intra-month
  pain (a put-assignment cascade into the crash). Worst single day -9.14%, worst
  5-day -15.99%. This is precisely the failure mode flagged in I2's caveats, now
  quantified: an 8x understatement.

* APPLES-TO-APPLES vs A DAILY-MARKED INDEX. The "~1/4 the drawdown" framing in I2
  was an artifact of comparing the wheel's MONTHLY DD to the index's continuous DD.
  Marked the SAME WAY (daily), the wheel/index drawdown ratio is:
      crash_2020          0.61   (wheel -20.56% vs cap-index -33.83%)
      bear_2022           0.38   (wheel -10.31% vs cap-index -27.41%)
      recovery_2023_2024  0.40   (wheel  -4.03% vs cap-index -10.07%)
  The wheel DOES take materially less drawdown than the index (~0.4-0.6x), and the
  defensive edge is real -- but it is NOT the ~0.1-0.25x that the monthly-vs-daily
  mismatch implied. In the crash it took 0.61x the index drawdown, not a quarter.
  (Index cap proxy cross-checked vs reality: COVID trough -33.83% on 2020-03-23 and
  2022 bear -27.41% on 2022-10-14 both match the actual S&P 500 path.)

* SHARPE/SORTINO collapse under daily marks (as expected -- daily vol is visible):
      regime               daily_Sharpe / Sortino   I2_monthly_Sharpe(reproduced)
      crash_2020              0.70 / 0.66                    1.78
      bear_2022              0.27 / 0.30                    0.53
      recovery_2023_2024     2.71 / 3.17                    3.66
  Monthly Sharpe was 2-2.5x the daily Sharpe in the two stress regimes by smoothing
  away intra-month vol. Worst-1d / worst-5d tails (crash: -9.14% / -15.99%) are severe.

OPTION-MTM APPROXIMATION & BIAS DIRECTION
  Both this daily mark and I2's monthly mark value open options at INTRINSIC ONLY
  (put liab = max(0,K-S)*100, call liab = max(0,S-K)*100), ignoring time value.
  A short option's true buyback cost = intrinsic + time value >= intrinsic, so
  intrinsic-only UNDERSTATES the liability and marks NAV slightly HIGH. Therefore
  the true daily drawdowns are, if anything, marginally DEEPER than reported here
  -- these are conservative LOWER bounds. (Using intrinsic-only is also what makes
  daily == monthly exactly at snapshot dates, which is why the reproduction is exact.)
  rf-on-collateral is accrued daily on the same liquid base I2 uses.

  Second-order caveats: index proxy is current-members-only (survivorship ->
  benchmark biased UP, so the wheel's relative drawdown edge is if anything
  understated); index DD is price-only (dividends omitted -> immaterial to DD shape);
  ~30-45% of option fills are modeled-spread fallbacks (inherited from I2); in
  crash_2020 the wheel is undeployed until the first in-window snapshot 2020-02-03,
  so the -20.56% is measured from a flat-NAV start (faithful: no positions = no risk).

BOTTOM LINE: I2's regime DIRECTION holds (wheel is defensive, beats index in
down/sideways tape on drawdown), but I2's specific drawdown MAGNITUDES are lower
bounds that understate true path risk by 1.6x-8x. A risk officer should size the
wheel against the DAILY numbers: a -20% intra-crash drawdown, not -2.6%.
